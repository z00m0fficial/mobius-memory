# Mobius Memory Architecture

Mobius Memory is the Organizational Memory Service for Mobius.

## Boundary

Mobius Memory stores verified organizational knowledge. It does not own AI routing, voice capture, metrics calculation, or the user interface.

## Inputs

- Sonic voice intelligence
- Router decisions
- Atlas recommendations
- Repository updates
- Platform Change Proposals
- Pulse metric snapshots

## Outputs

- Memory events
- Organizational timeline
- Knowledge graph records
- Search results
- Executive briefing context

## Verification Rule

Memory can be stored as raw organizational history, but it cannot be reused as trusted decision context until verified.
